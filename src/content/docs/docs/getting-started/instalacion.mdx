---

import { Tabs, TabItem } from "@astrojs/starlight/components";

tabus-js no tiene dependencias en tiempo de ejecución. Instálalo con el gestor de paquetes que uses en tu proyecto:

<Tabs>
<TabItem label="pnpm" icon="pnpm">
```bash
pnpm add tabus-js
```
</TabItem>
<TabItem label="npm" icon="npm">
```bash
npm install tabus-js
```
</TabItem>
<TabItem label="yarn">
```bash
yarn add tabus-js
```
</TabItem>
<TabItem label="bun" icon="bun">
```bash
bun add tabus-js
```
</TabItem>
</Tabs>

El paquete incluye sus propios tipos de TypeScript: no hace falta instalar `@types/tabus-js` por separado.
